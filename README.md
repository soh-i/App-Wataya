# WatayaSan
WatayaSan notifies a newly published book written by 綿矢りさ to send your OSX Notification center.

## INSTALLATION
To install this module, run the following commands:

```
perl Makefile.PL
make
make test
make install
```
        
## Usage
`$ ./wataya`
